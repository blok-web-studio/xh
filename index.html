Built for Humans

Everything you need.
Nothing you don't.

xh rethinks HTTP from the terminal up — colour, structure, and zero unnecessary ceremony.

JSON by default
Every response is automatically formatted and syntax-highlighted. No piping to jq required. Headers, body, status — all colour-coded at a glance.
Native Auth
Bearer tokens, basic auth, API keys — xh handles them inline. Authorization:bearer $TOKEN just works.
Persistent Sessions
Log in once, reuse across requests. Sessions persist cookies and headers so you don't have to replay auth every time.
Colourised Output
Every response type has its own colour scheme. Headers in cyan, keys in blue, strings in green — your terminal becomes readable instantly.
Download Support
Download files directly from the terminal. Progress bars, resume support, and automatic filename detection included.
Blazing Fast
Written in Rust, built for speed. No Node dependency, no Python overhead. A single static binary, zero setup.

Get Started

Install in one line.

Works on macOS, Linux, and Windows. One binary, zero dependencies.

macOS (Homebrew)
brew install xh
Cargo
cargo install xh
Linux (APT)
snap install xh
Nix
nix-shell -p xh
macOS / Linux (binary)
curl -fsSL https://github.com/ducaale/xh/releases/latest/download/xh-$(uname -s)-$(uname -m).tar.gz | tar xz && sudo mv xh /usr/local/bin
