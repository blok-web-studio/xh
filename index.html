$ xh get /compare

One POST request. Two experiences.

curl — 16 flags
curl -s -H "Accept: application/json" \
-H "Authorization: Bearer $TOKEN" \
-X POST https://api.example.com/data \
-d '{"name":"xh","type":"tool"}' \
| jq .
// still debugging
xh — 0 flags
xh POST https://api.example.com/data \
name=xh type=tool \
Authorization:"Bearer $TOKEN"

✓ HTTP/2.0 201 — readable. coloured. done.
